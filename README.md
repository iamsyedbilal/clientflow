## 📦 Installation

Clone the repository.

```bash
git clone https://github.com/iamsyedbilal/clientflow.git
```

Move into the project.

```bash
cd clientflow
```

Install dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
